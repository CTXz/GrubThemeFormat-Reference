## selected_item_color
<b> <sup> Data Type : Font </sup> </b>

Stores font for inactive/unselected menu item texts

If undefined : Unknown Regular 16
#
In practice :

```
# -- theme.txt --

# Display menu items text in DejaVu Sans Regular, size 14
+ boot_menu {
	item_color = "DejaVu Sans Regular 14"
}
```

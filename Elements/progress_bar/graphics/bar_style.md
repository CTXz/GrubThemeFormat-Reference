## bar_style
<b> <sup> Data Type : Styled Box </sup> </b>

Defines progress bar background texture (decreasing bar)
If undefined : set to [bg_color](bg_color.md)
#
In practice :

```
# -- theme.txt --

# Set progressbar background texture to bg_bar_*.png
+ progress_bar {
	bar_style = bg_bar_*.png
}
```
